# archserver
servidor caseiro arch linux

## 🎯 Intuito do Projeto

Este repositório documenta a criação e a administração de um **servidor Arch Linux** dedicado a estudos e testes. O ambiente serve como um laboratório pessoal para experimentar configurações, serviços, scripts e práticas de administração de sistemas Linux. Todas as alterações, melhorias e aprendizados são registrados aqui, servindo como um guia prático e um portfólio do meu aprendizado em infraestrutura Linux.
